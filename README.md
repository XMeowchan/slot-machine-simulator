# 🎰 老虎机模拟器

一个交互式的老虎机游戏，用于展示赌场数学模型和概率设计。

## 🎮 在线体验

[点击这里玩游戏](https://xmeowchan.github.io/slot-machine-simulator/)

## ✨ 特性

- 🎰 真实的旋转动画效果
- 💰 可调节下注金额（¥10/20/50/100）
- 📊 实时显示余额、旋转次数、RTP
- 🎉 不同中奖类型的视觉反馈
- 🔥 Near-miss 效应演示
- 📋 完整的赔付表

## 🎲 游戏机制

- 初始余额：¥1000
- RTP（玩家回报率）：约 93%
- 赌场优势：约 7%
- 基于真实的虚拟卷轴概率设计

## 📚 教育目的

这个项目展示了：
- 老虎机的数学模型
- 虚拟卷轴技术
- Near-miss 心理效应
- 长期来看玩家必输的数学原理

## 🚀 本地运行

直接在浏览器中打开 `slot-game.html` 即可。

## 📖 相关文件

- `slot-game.html` - 可交互的网页游戏
- `slot-machine-simulator.js` - 命令行模拟器
- `realistic-slot.js` - 真实赌场级别模拟
